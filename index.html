AI Japanese Learning Student Demo

This is a student-side prototype for AI-assisted Japanese learning. It demonstrates practice, feedback, and progress review in a simple interface.

Start Demo

Project Purpose

This prototype shows how students can learn Japanese through interactive tasks and receive immediate feedback after answering questions.

Main Functions

  • Basic grammar exercises
  • Sentence ordering tasks
  • Fill-in-the-blank practice
  • Immediate feedback display
  • Simple learning progress tracking

Learning Flow

Learn → Answer → Receive Feedback → Review Progress
